# so_long
This project is a small 2D game with minilibx.

## How to use

Clone the project
```bash
git clone https://github.com/thomasbrq/so_long.git
```

Go to project directory
```bash
cd so_long
```

Build the executable.
```bash
make
```

Then launch the program
```bash
./so_long maps/<name>.ber
```
